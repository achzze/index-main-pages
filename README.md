# index-main-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能用于展示、测试、演示、文档或其它通用用途。仓库本身不针对任何特定域名、站点或项目，而是作为一个集中存储和版本维护的场所。

## 项目简介

`index-main-pages` 是一个通用型的 HTML 页面存档仓库。所有页面均为独立的 HTML 文件，彼此之间无强依赖关系。每个页面可以单独运行，无需额外服务器或数据库支持。本仓库的主要目的是方便地存储、管理和共享这些页面，同时利用 GitHub 的版本控制功能追踪历史变更。

## 目录说明

仓库根目录下直接存放各类 HTML 页面文件。为保持结构清晰，建议按页面用途或主题进行组织，例如：

- `pages/` — 存放主要页面
- `archive/` — 存放较旧或不再活跃的页面
- `demo/` — 存放演示或示例页面

你也可以根据需要自行创建子目录。每个目录内可附带 `README.md` 文件说明该目录下的页面内容。

## 页面归档说明

- 所有页面均为纯 HTML，不依赖外部资源（或资源已内联）。
- 页面文件名应具有描述性，便于识别内容。
- 建议在每个页面内添加注释，说明页面的用途、创建日期或版本信息。
- 提交时请附上清晰的 commit 信息，说明新增或修改的内容。

## 维护说明

- 欢迎提交 Pull Request 添加新的页面或改进已有页面。
- 请确保新增页面不会破坏现有页面功能。
- 对于长期未更新的页面，可考虑移入 `archive` 目录。
- 本仓库不托管敏感信息或需要登录才能访问的资源。
- 如有疑问，请通过 Issues 进行讨论。
